# SlideBot

AI-Powered PowerPoint Slides Generator.

Built to simplify presentation creation by using AI to transform topics into structured, professional slide decks.

---

## Tech Stack
- **Framework:** Python
- **AI Integration:** [Gemini/GROQ API]


---

## About
SlideBot is an automated tool that generates presentation slides from text prompts, reducing the time required to create professional slide decks.
